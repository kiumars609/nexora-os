# Nexora OS

**Nexora OS** is an experimental **next-generation gaming operating system simulation** built for the web.
It explores **system-driven UI**, **controller-first UX**, and **scalable frontend architecture** — without copying any existing console UI.

> Status: **v0.1 — Initial setup**

---

## Vision
Build a brand-new console-like OS experience that feels:
- **Fast** and focused (gaming-first)
- **System-like** (apps, states, lifecycle)
- **Controller-friendly** (keyboard/gamepad navigation)
- **Original** (not a PlayStation/Xbox clone)
